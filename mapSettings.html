Map Settings

These settings adjust the appearance of map features.


Maximum Flow Width: Adjusts the maximum pixel width of flows. The flow with the highest value will be the maximum width, and the width of the other flows will be proportional to it based on their values. Click the eye icon next to the slider to hide/show flows.

Maximum Node Size: Adjusts the maximum pixel radius of start and end nodes. The node with the highest value will have the maximum radius, and the area of the other nodes will be proportional to it based on their values. Click the eye icon next to the slider to hide/show flows.

Lock Feature Sized to Current Scale checkbox: Checking this box will cause map features to be sized relative to the current scale (aka, features will shrink as the map is zoomed out). Unchecking this box will cause features to appear the same size no matter what the zoom level is. This box is checked automatically when opening data from CSV files or settings.

Map Layers: This window will display the names of any shapefiles that have been added to the map. The draw order of layers can be changed by draging layers to different locations on the list. Layers can be added or removed by clicking the plus and minus icons, as well as through the File menu. The fill and stroke of selected layers can be adjusted using the color pickers just below the window.

Move Flows That Cross Nodes: Clicking this button can improve the map layout by identifing flows on the map that touch unrelated nodes and move them off the nodes.
