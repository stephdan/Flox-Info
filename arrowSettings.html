Arrow Settings

These settings adjust the appearance of flow arrows.


Add Arrows: Check this box to add arrowheads to the ends of flows. The arrow will point to the destination node.

Flow Distance From End Point: The distance in pixels that the tip of the arrow is from the destination node. Enter a new value into the text box to change it. If flows are being clipped to polygons, this will be the distance of the tip of arrow from the border of the clipping polygon.

Arrowhead Length: Adjusts the length of the arrowheads.

Arrowhead Width: Adjusts the width of the arrowheads.

Arrow Edge Ctrl Point Length: Changes the location of the control point defining the side edges of arrows. Higher values move the control point towards the tips of arrows.

Arrow Edge Ctrl Point Width: Changes the location of the control point defining the side edges of arrows. Higher values move the control point away from the arrow perpendicular to the arrow's center line.

Arrow Corner Position: Changes the locations of the side corners of the arrows. Higher values move the corners towards the tip of the arrows.

Arrow Size Ratio: Arrows are automatically sized relative to the values of their flows. This slider will adjust the relative size of the smallest arrowheads. The highest value of this slider will make all arrows the same size as the largest arrow reguardless of the width of their flows.
