Quick Tutorial

This short tutorial will get you started with Flox.

If you haven't yet downloaded the application, please refer to the Manual section for installation instructions.

Also, download this sample data, which you will need for the tutorial. This is a .zip file containing a sample shapefile and CSV flow data. Extract the .zip into a folder on your hard drive

Step 1: Opening Data

Start up Flox by double-clicking Flox.exe. When Flox first starts, you will be prompted to open a CSV Flow File. Navigate to the folder where you just extracted the sample data and open sampleData.csv using the dialog. Flows will appear on the screen and arrange themselves according to the default layout settings.

Next, select File>Add Shapefile Layer from the menu bar at the top of the window. Using the resulting dialog, navigate to the sample data folder and open the file with the .shp extention named countries_millerCylindrical_clipped.shp. A polygon layer of some European contries should appear.

Step 2: Modify Layout Settings

Select the Layout tab in menu at the right side of the window. You'll see several slider bars for changing the way flows are laid out on the map. For a description of what each slider does, see the Layout section of the Manual.

For now, try sliding the top-most slider labeled Stiffness of Longest Flow all the way to the left to 0. This will cause the flows to initiate the layout process. A progress bar will appear towards the bottom of the menu while the flows are being layed out.

Adjusting any of the sliders will initialize the layout process. Take some time to experiment with each one to see how it affects the layout.

The purpose of this layout process is to prevent excessive intersections and overlapping of flows in order to make the map easier to read. Optimal layout settings will depend on the arrangement and density of the flow data. The default settings are often sufficient, but in many cases they will need some tweaking.

Try adjusting the Layout settings until the flows seem to be arranged in such a way that intersections between flows are reduced and the flows are easy to tell apart.

Step 3: Manual Flow Adjustments

Sometimes the layout settings won't be able to arrange every flow better than you can. In these cases, individual flows can be adjusted manually using the Select/Move tool, which is the arrow icon at the far left of the toolbar at the top of the window. Click it now to activate the tool.

Clicking a flow line with this tool active will select the flow, making the small yellow control point visible. Click and drag the control point to change the shape of the flow.

Moving the control point of a flow automatically "locks" the flow. A locked flow will not move when layout settings are changed, however it will continue to have an influence on the location of other flows. Selected flows can be locked and unlocked using the Lock/Unlock Flows button in the toolbar.

Another tool is the Move Flows That Cross Nodes button at the bottom of the Layout tab. Pressing this can improve the layout.

Step 4: Creating Flow Data

You can create new flow data using the Add Flow Tool to add nodes and flows to the map, and you can use the Select/Move Tool to move existing nodes. Attributes can be added to the flows and nodes using the text boxes at the top of the window. See the Manual section for details about adjusting the map appearance and exporting the finished map and flow data.
