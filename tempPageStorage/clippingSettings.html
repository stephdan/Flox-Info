Clipping Settings

Flows can be clipped to polygon areas using this menu. Currently only shapefile polygons can be used for clipping.


Clip Areas and Select Shapefile button: Click the Select Shapefile button to import he polygon shapefile layer you'd like to clip flows to.


Clip with End Areas checkbox: Checking this box will clip flows at the border of the polygons they enter.

End Area Buffer Distance: Enter a value into this checkbox to add an inward buffer to the clipping polygons.

Draw checkbox: Check this box to view the clipping polygons (with buffer applied).


Clip with Start Areas checkbox: Checking this box will clip flows at the border of the polygons they leave.

Start Area Buffer Distance: Enter a value into this checkbox to add an inward buffer to the clipping polygons.

Draw checkbox: Check this box to view the clipping polygons (with buffer applied).
