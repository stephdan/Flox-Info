Settings

The right side of the window has controls for adjusting the layout forces, map appearance, arrows, and flow clipping properties.

  • Layout
  • Map
  • Arrows
  • Clipping

Data Import and Export

Access data importing/exporting options by clicking the File menu item at the top left of the window.

  • Open Flows
  • Open Nodes and Flows
  • Open/Save Settings
  • Add Shapefile Layer
  • Export SVG, CSV, Image

Toolbar

The toolbar at the top of the window can be used to edit flows and navigate the map.

Select/Move Tool, used for selecting and moving map features.

Add Flow Tool, used for adding new flows to the map.

Button for locking/unlocking selected flows (icon changes based on the status of the currently selected flow).

Tools for zooming and panning around the map.

Adjusts the zoom level to show all loaded data sets.

Measures distances.
