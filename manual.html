Toolbar

The toolbar at the top of the window can be used to edit flows and navigate the map. Some of the functionality of the toolbar is also available in the menu items (such as selecting and deleting flows).

Select/Move Tool: Used for selecting and moving map features. With this tool active, click a flow or node to select it, or click and drag an empty space to select features by rectangle. Selected nodes can be dragged to any location on the map. A selected flow will display the location of its control point, which can then be dragged to a new location to manually adjust the curve of the flow.

Flows will automaticaly lock once they are manually adjusted. Selected flows and nodes can be deleted with the delete key

Add Flow Tool: Used for adding new flows to the map. With this tool active, click on an existing node or an empty space to select/create a start point, and then click another existing node or empty space to select/create an end point. A flow will be drawn connecting the two points.

Lock/Unlock button: Locks or unlocks selected flows. The icon changes based on the status of the currently selected flow(s)). Locked flows retain their current shape when making adjustments to the layout settings.

Zoom and Pan Tools: Used for map navigation.

Show All: Adjusts the zoom level to show all loaded data sets.

Measure Tool: With this tool active, click and drag the map to measure map distances.

Additional Flow Editing Options

Editable Text Boxes: The top of the window has 3 editable text boxes that display the value and x/y coordinates of any selected feature. These attributes can be edited by entering new values into these boxes while a feature is selected.

The X and Y boxes will be blank and uneditable if a flow or more than one node is selected. However, the Value box may still be edited while multiple features of any type are selected, and the new value will be attributed to all selected features.

Menu Bar: Some of the map editing and navigation functionality is available in the menu bar at the top of the window, as well as some options that are not available elsewhere.
