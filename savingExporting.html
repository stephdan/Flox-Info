Saving and Exporting Data

There are several options for saving and exporting map data.


Save Settings: Saves the current application settings and flow data into an XML file that can be reopened using File>Open Settings.

Export SVG: Exports all flow and shapefile data into an SVG file, which is openable by vector editing software like Adobe Illustrator, as well as most web browsers.

Export CSV: Exports the flow data into a Comma Separated Value text file. This is useful for importing the data into table editing software like Microsoft Excel. Does not include shapefile data.

Export Image: Exports a png image of the currently visible map area. When selected, a menu will pop up where you can change the length of the longest side of the image in pixels, allowing you to resize the image. The default value is the current size of the map window.


Importing Data

Flox can import specific types and formats of data.


Open Flows: Opens a dialog for importing flow data from a Comma Separated Value (CSV) file. The CSV needs to be formatted in a specific way for Flox to be able to read it. It must include five columns with 3 additional optional columns in the following order without headers:

  • Start point x coordinate
  • Start point y coordinate
  • End point x coordinate
  • End point y coordinate
  • Flow value
Optional Columns
  • Control point x coordinate
  • Control point y coordinate
  • Lock status (0 = unlocked, 1 = locked)

Open Nodes and Flows: Opens a dialog for importing flow data from two Comma Separated Value (CSV) files, one containing node coordinates and the other containing the connectivity between those nodes. The dialog includes two buttons for browsing to your data.

  • Nodes File: This file should be a CSV with 3 columns in the following order without headers:
    • Node ID (can be numbers or letters)
    • X coordinate
    • Y coordinate

  • Flows File: This file should be a CSV file with 3 columns in the following order without headers:
    • Start Node ID (should match a node in the Nodes File)
    • End Node ID (should match a node in the Nodes File)
    • Flow Value (a number)

Open Settings: Opens the settings XML file that is produced by the Save Settings option.


Open Shapefile: Adds an Esri shapefile point or polygon layer to the map. When browsing to the location of the shapefile, select the file with the .shp extension.
